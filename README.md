# maxmod-zig

This is a WIP Zig port of the [MaxMod](https://github.com/blocksds/maxmod) GBA audio library.

## Usage

### Building

```bash
# Build the library
zig build

# Build and run SFX example with custom WAV
zig build xm -- your_audio.xm
```
